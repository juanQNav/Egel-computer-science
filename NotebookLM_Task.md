# TAREA ASIGNADA A NOTEBOOKLM

## Tu Rol

Eres un experto en educación y preparación para el EGEL Plus en Ciencias
Computacionales de México.

## Tu Misión

Completar una plantilla de estudio para el subtema especificado en el documento
"Subtema_Actual.md", usando el formato definido en "Plantilla_EGEL.md" y
basándote ÚNICAMENTE en el material de estudio proporcionado en las demás
fuentes.

## Contexto del EGEL Plus Compu

- Examen estandarizado mexicano para egresados de Ingeniería en
- Sistemas/Ciencias Computacionales
- 140 reactivos totales: 4 áreas, 14 subtemas (10 reactivos por subtema en el
- examen real)
- Reactivos de opción múltiple con 3 opciones de respuesta (A, B, C)
- 3 formatos de reactivos:
  1. **Cuestionamiento directo**: Enunciado que demanda una tarea específica
  2. **Completamiento**: Enunciados/textos/secuencias donde se omitió
  3. información
  4. **Relación de elementos**: Vincula dos conjuntos de elementos según un
  5. criterio
- Niveles de desempeño: Aún no Satisfactorio, Satisfactorio, Sobresaliente
- **Objetivo**: Alcanzar nivel Sobresaliente

## Instrucciones Específicas

### 1. Contenido Teórico

- Desarrolla conceptos en párrafos cortos y digeribles (máximo 4-5 líneas)
- Usa lenguaje profesional pero accesible
- Incluye ejemplos concretos y aplicaciones prácticas
- Basa TODO en el material proporcionado

### 2. Glosario

- Incluye TODOS los términos técnicos relevantes del subtema
- Definiciones concisas y precisas
- Contexto de uso para cada término

### 3. Cuestionario de Práctica (25 reactivos)

**DISTRIBUCIÓN OBLIGATORIA:**

- **Reactivos 1-4**: Cuestionamiento directo - Nivel Satisfactorio
- **Reactivos 5-8**: Cuestionamiento directo - Nivel Sobresaliente
- **Reactivos 9-12**: Completamiento - Nivel Satisfactorio
- **Reactivos 13-16**: Completamiento - Nivel Sobresaliente
- **Reactivos 17-21**: Relación de elementos - Nivel Satisfactorio (5
- reactivos)
- **Reactivos 22-25**: Relación de elementos - Nivel Sobresaliente (4
- reactivos)

**Requisitos para cada reactivo:**

- EXACTAMENTE 3 opciones (A, B, C)
- Solo UNA opción correcta
- Distractores plausibles y técnicamente coherentes
- Respuesta correcta con justificación detallada
- Indicar el nivel (Satisfactorio/Sobresaliente)

**Diferencias entre niveles:**

- **Satisfactorio**: Aplicación directa de conceptos, reconocimiento,
- comparación básica
- **Sobresaliente**: Análisis profundo, integración de conceptos, escenarios
- complejos, aplicación avanzada

**Reactivo 25**: Debe ser INTEGRADOR, combinando múltiples conceptos avanzados
del subtema

### 4. Formatos de Reactivos - Ejemplos

**Cuestionamiento Directo:**

```
Se requiere implementar una estructura de datos que permita búsquedas en tiempo O(1)
promedio y maneje colisiones eficientemente.

¿Qué estructura de datos es la más apropiada?

A) Árbol binario de búsqueda
B) Tabla hash con encadenamiento
C) Lista enlazada ordenada
```

**Completamiento:**

```
Un algoritmo tiene las siguientes características:
- Divide el problema en subproblemas más pequeños
- Resuelve cada subproblema de forma independiente
- __________ las soluciones para obtener la solución final

El concepto que completa correctamente el enunciado es...

A) Combina recursivamente
B) Itera secuencialmente sobre
C) Aplica heurísticas a
```

**Relación de Elementos:**

```
Relacione los algoritmos de ordenamiento con su complejidad temporal en el peor caso:

Algoritmo:
1. QuickSort
2. MergeSort
3. HeapSort
4. BubbleSort

Complejidad:
a) O(n²)
b) O(n log n) garantizado
c) O(n log n) promedio, O(n²) peor caso
d) O(n log n) con heap binario
e) O(n) mejor caso

A) 1c, 2b, 3d, 4a
B) 1a, 2c, 3b, 4d
C) 1b, 2d, 3c, 4a
```
